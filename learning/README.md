# Learning Resources

This directory contains educational materials, tutorials, and step-by-step guides for learning AI concepts and implementations.

## Structure

- **fundamentals/** - Core AI/ML concepts and basics
- **llm-workflows/** - Learning materials for Large Language Model workflows
- **embeddings/** - Understanding and working with embeddings
- **rag/** - Retrieval-Augmented Generation tutorials
- **agents/** - AI agent patterns and implementations
- **mcp-servers/** - Model Context Protocol server examples

## Getting Started

Each subdirectory contains:
- Conceptual explanations
- Code examples
- Exercises and challenges
- Additional resources and references

## Contributing

When adding new learning materials:
1. Follow the existing structure
2. Include clear explanations and comments
3. Provide working code examples
4. Add prerequisites and setup instructions
5. Include expected outputs or results
