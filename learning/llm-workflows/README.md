# LLM Workflows

Learning materials for working with Large Language Models, including prompting, chaining, and workflow orchestration.

## Topics Covered

- Introduction to LLMs
- Prompt engineering
- Chain-of-thought prompting
- Few-shot learning
- Function calling
- Streaming responses
- Context management
- Token optimization
- Error handling and retries
- LLM orchestration patterns

## Examples

This section includes:
- Basic LLM API interactions
- Advanced prompting techniques
- Building conversational flows
- Multi-step reasoning examples
- Workflow orchestration patterns

## Supported Providers

Examples may include:
- OpenAI (GPT-3.5, GPT-4)
- Anthropic (Claude)
- Google (Gemini)
- Open-source models (Llama, Mistral)

## Prerequisites

- Understanding of API calls
- Basic knowledge of async programming
- Familiarity with JSON
